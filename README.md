# collect

**a form creation and submission api**

## Setup

- setup a cockraochb instance
- `export DATABASE_URL = <url>`
- `pip install -r requirements.txt`
- `cd src` and  `uvicorn main:app`
- `python3 demo.py`
- `cd demo` and `npm i` and then `node demo.js`
